## Troubleshooting

### Can't see the heatmap?
- Check if you have Premium subscription
- If not, tap "Upgrade to Premium" button
- Free users see a teaser instead

### Heatmap looks empty?
- Start completing your habit daily
- It takes a few days to see patterns
- Gray squares = not completed yet

### Arrow button not visible?
- Make sure you're on the latest version
- The arrow button is on the right side of each habit card
- It's next to the bell (reminder) icon
