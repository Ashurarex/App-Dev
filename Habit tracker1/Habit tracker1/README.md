# HabitTracker (Improved UI)
This is an improved UI version scaffold for the HabitTracker Flutter app.

## Features
- Firebase Auth (email + Google scaffold + phone-ready)
- Firestore for habits CRUD
- Improved responsive UI: grid layout for wider screens
- Dark mode toggle persisted with SharedPreferences
- Animations using flutter_animate
- Drawer navigation, modal forms, slidable actions

## Setup
1. Create Firebase project and add Android/iOS/Web app.
2. Add google-services.json / GoogleService-Info.plist / web config.
3. Enable Auth (Email, Phone, Google) and Firestore in Firebase console.
4. Run `flutter pub get` then `flutter run`.

Note: Replace any placeholder values and wire Firebase config files before running.
